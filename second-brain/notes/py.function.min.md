la fonction `min()` retourne l’élément parmis ceux passés en argument ayant la 
plus petite valeur.

```python
x = min(5, 10)
```

Dans ce cas, `x` est affecté de la valeur 5.

On peut rapidement retrouver la valeur minimale d’une liste :  

```python
scores = [3, 6, 4, 12]
print(min(scores))
```

```shell
3
```
