La commande `script` permet d’enregistrer les inputs et outputs de la session terminal.

```shell
$ script 03-21-2021-0200pm-exploitation.log
```

À la fin de la session, il s’uffit d’`exit` le script, comme ceci :

```shell
$ exit
```
