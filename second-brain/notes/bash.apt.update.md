Cette commande permet de vérifier les *packages* pour lesquels une mise-à-jour et/ou une nouvelle versions existent.


```shell
$ apt update 
```

Pour procéders à la mise-à-jour, on doit utiliser la commande [[bash.apt.upgrade]]
