La fonction `lambda` permet de créer, en python, des fonctions mathématiques telles que :

$$
f(x) = 2x
$$

La syntaxe est :

```python
f = lambda x: 2 * x
print(f(3))
```

Dans ce cas `print(f(3))` retourne `6`.

```python
f = lambda x, y: 2 * x + y
print(f(3, 2))
```

Dans ce cas `print(f(3))` retourne `8`.
