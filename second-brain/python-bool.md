In python, [[boolean]] syntax is:

- `True`
- `False`

```python
variable = True
type(variable)
```

## `bool()`

```python
print(bool(0))
>>> False
```

```python
print(bool(1))
>>> True
```

```python
print(bool('Hello!'))
>>> True
```
