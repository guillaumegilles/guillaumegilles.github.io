la chaîne de caractères, `"hello"`, est un type de donnée *character*. abbréviée 
`chr`.

```{r}
> x <- "hello"
> mode(x)
[1] "character"
```

On peut tester l’appartenance un [[mode|r.dt]] `character`grâce à la fonction 
[[r.base.is-character]]


```{r}
> is.character(x)
[1] TRUE
```
