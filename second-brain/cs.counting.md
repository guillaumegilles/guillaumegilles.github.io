Par convention en programmation, lorsqu'on doit implémenter un morceau de code pour compter, on utilise la variable ´i´ pour *integer*.

```c
int i = 0;
while (i < 3)
{
    printf("meow\n");
    i++;
}
```

Si on doit utiliser une variable supplémentaire, on utilise la lettre `j`.
