`Math` est un package pour faciliter les calculs math√©matiques, comme :

$$
C = 2 \pi r
$$

$$
A = \pi r^2
$$

```python
# Definition of radius
r = 0.43

# Import the math package
import math

# Calculate C
C = 2 * math.pi * r

# Calculate A
A = math.pi * r * r

# Build printout
print("Circumference: " + str(C))
print("Area: " + str(A))
```
