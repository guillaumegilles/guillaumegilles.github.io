[[Les opérateurs de comparaison|cs.comparison-operator]] en SQL possède une syntaxe un peu particulière :

| Opérateur           |       SQL       |
| :------------------ | :-------------: |
| Égalité             |       `=`       |
| Inégalité           |      `<>`       |
| Inférieur à         |       `<`       |
| Inférieur ou égal à |      `<=`       |
| Supérieur à         |       `>`       |
| Supérieur ou égal à |      `>=`       |
