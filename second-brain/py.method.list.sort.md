Avec Python, on peut ordonner une liste python de façon ascendante, grâce à la [[méthode|py.method]] `sort()` :

```python
scores = [3, 6, 1, 12]
scores.sort()
print(scores)
```

```shell
[1, 3, 6, 12]
```
