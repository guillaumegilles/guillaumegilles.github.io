## `range()`

In [[python]], the `range()` is a useful function to write [[python-for-loops]].

```python
for i in range(3):
    print("Doing important work. i =", i)
>>> Doing important work. i = 0
>>> Doing important work. i = 1
>>> Doing important work. i = 2
```
