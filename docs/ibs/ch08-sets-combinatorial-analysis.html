Definition

A set is the mathematical model for a collection of different and distinct things / objects ; a set contains elements or members, which can be mathematical objects of any kind: numbers, symbols, points in space, lines, other geometrical shapes, variables, or even other sets. The set with no elements is the empty set; a set with a single element is a singleton. A set may have a finite number of elements or be an infinite set.

Definitions : A set рқҗё is a collection of objects, called its elements, considered without order or possible repetition (parce que ce sont des objets). Combinatorial analysis

The purpose of combinatorial analysis (counting techniques) is to learn how to count the number of elements in a finite set.

Three techniques will be addressed: Permutations Arrangements Combinations

These techniques depend on an operation: the factorial of a nonnegative integer.

The factorial :Let рқ‘ӣ вҲҲ рқ‘Ғ. Its factorial is defined by :

рқ‘ӣ! = 1 Г— 2 Г— . . . Г— (рқ‘ӣ вҲ’ 1) Г— рқ‘ӣ

By convention, we have 0! = 1. Main Property: Let рқ‘ҳ be a nonnegative and non null integer (рқ‘ҳ вүҘ 1) and let рқ‘ӣ be a nonnegative and non null integer such that (рқ‘ӣвҲ’рқ‘ҳ)вүҘ0.

We have : рқ‘ӣ! = (рқ‘ӣ вҲ’ рқ‘ҳ)! Г— (рқ‘ӣ вҲ’ рқ‘ҳ + 1) Г— . . . Г— (рқ‘ӣ вҲ’ 1) Г— рқ‘ӣ.

Permutations :Given a set рқҗё of рқ‘ӣ objects, a permutations is an ordered rearrangement, without repetition of these рқ‘ӣ distinct objects. The number of permutations of рқ‘ӣ objects equals рқ‘ӣ!

Arrangements : Given a set рқҗё of рқ‘ӣ objects (elements), we call arrangements of рқ‘қ (1 вүӨ рқ‘қ вүӨ рқ‘ӣ) objects, all ordered sequences of рқ‘қ objects taken from the рқ‘ӣ objects.

There are two cases:

Arrangements without repetition : When each object can only be seen once in an arrangement, the number of non-repeating arrangements of рқ‘қ objects taken from рқ‘ӣ is: рқҗҙ_рқ‘ӣ^рқ‘қ= рқ‘ӣ!/((рқ‘ӣвҲ’рқ‘қ)!) where 1вүӨрқ‘қвүӨрқ‘ӣ.

Arrangements with repetition : When an object can be observed several times in an arrangement, the number of arrangements with repetition of рқ‘қ (1 вүӨ рқ‘қ вүӨ рқ‘ӣ) objects taken from рқ‘ӣ is equal to рқ‘ӣ^рқ‘қ.

Remark : The permutation of рқ‘ӣ objects is a particular case of a non-repeating arrangement of рқ‘қ objects taken from рқ‘ӣ objects, when рқ‘қ = рқ‘ӣ.

Thus, the number of permutations of рқ‘ӣ objects equals :

рқҗҙ_рқ‘ӣ^рқ‘ӣ= рқ‘ӣ!/((рқ‘ӣвҲ’рқ‘ӣ)!)=рқ‘ӣ!/0!=рқ‘ӣ!

Combinations : Given a set рқҗё of рқ‘ӣ objects, we call combinations of рқ‘қ (1 вүӨ рқ‘қ вүӨ рқ‘ӣ) objects any set of рқ‘қ objects taken  without replacement among the рқ‘ӣ objects. In this case, the notion of order of objects is no longer taken into account. The number of combinations of рқ‘қ objects taken from рқ‘ӣ is

рқҗ¶_рқ‘ӣ^рқ‘қ= рқ‘ӣ!/(рқ‘қ! Г— (рқ‘ӣвҲ’рқ‘қ)!)=(рқҗҙ_рқ‘ӣ^рқ‘қ)/рқ‘қ!
