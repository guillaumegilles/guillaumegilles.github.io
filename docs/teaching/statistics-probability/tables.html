Table de la loi normale \(\mathcal{N}(0,1)\)

Mode d’emploi

La table donne \(\Phi(z) = P(Z \leq z)\) pour \(Z \sim \mathcal{N}(0,1)\).

  • Symétrie : \(\Phi(-z) = 1 - \Phi(z)\).
  • Intervalle symétrique : \(P(-z \leq Z \leq z) = 2\Phi(z) - 1\).
  • Valeurs clés : \(\Phi(1{,}645) \approx 0{,}95\), \(\Phi(1{,}960) \approx 0{,}975\), \(\Phi(2{,}326) \approx 0{,}99\), \(\Phi(2{,}576) \approx 0{,}995\).

Lecture : pour \(z = 1{,}96\), lire la ligne \(z = 1{,}9\) et la colonne \(0{,}06\) : \(\Phi(1{,}96) = 0{,}9750\).

Table de la loi de Student

Mode d’emploi

La table donne le fractile \(t_{\nu,\, P}\) tel que \(P(T \leq t) = P\) pour \(T \sim \mathcal{T}(\nu)\) (loi de Student à \(\nu\) degrés de liberté).

Pour un test bilatéral au risque \(\alpha\), on lit la colonne \(P = 1 - \alpha/2\).

Exemple : pour \(\nu = 10\) degrés de liberté au niveau 95 % (test bilatéral, \(\alpha = 5\,\%\)), on lit la colonne \(P = 97{,}5\,\%\) : \(t_{10;\, 0{,}975} = 2{,}228\).
