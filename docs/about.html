About

From finance to firewalls — I secure what matters.

I’m Guillaume Gilles, a former Banque de France financial analyst who now channels a decade of risk assessment and teaching into cybersecurity.

For over ten years at the Banque de France, I decoded macroeconomic patterns, assessed corporate resilience, and produced strategic insight for decision-makers. My academic side ran parallel — teaching statistics, financial analysis, and information systems at top business schools, instilling analytical rigor and adaptability.

Hands-on with GitHub, I write, experiment, and explore: bridging data, defense, and human behavior to uncover and harden vulnerabilities.

Today, I’m immersed in cybersecurity, building skills in digital forensics, secure network design, and incident investigation. My passion lies at the nexus of data, security safeguards, and the behaviors that drive human error.

d3 = require("d3@7")

// Skill data
skills = [
  { name: "Python", value: 90, color: "#377eb8" },
  { name: "R", value: 75, color: "#4daf4a" },
  { name: "JavaScript", value: 65, color: "#ff7f00" },
  { name: "C++", value: 50, color: "#e41a1c" }
];

width = 500;
height = 150;
radius = 45;
thickness = 10;

// SVG container
svg = d3.create("svg")
  .attr("width", width)
  .attr("height", height);

// Group for each skill
g = svg.selectAll("g")
  .data(skills)
  .join("g")
  .attr("transform", (d, i) => `translate(${80 + i * 110}, ${height / 2})`);

// Background circle
g.append("circle")
  .attr("r", radius)
  .attr("fill", "none")
  .attr("stroke", "#e6e6e6")
  .attr("stroke-width", thickness);
arc = d3.arc()
  .innerRadius(radius - thickness)
  .outerRadius(radius)
  .startAngle(0);

// Skill arc
g.append("path")
  .attr("fill", d => d.color)
  .attr("d", d =>
    arc({ endAngle: (d.value / 100) * 2 * Math.PI })
  );
g.append("text")
  .attr("text-anchor", "middle")
  .attr("dy", "0.35em")
  .style("font-size", "14px")
  .style("font-weight", "bold")
  .text(d => `${d.value}%`);
g.append("text")
  .attr("text-anchor", "middle")
  .attr("dy", "2.2em")
  .style("font-size", "12px")
  .text(d => d.name);
